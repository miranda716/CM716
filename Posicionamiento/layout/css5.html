Curso de Introducción a CSS

CSS (Cascading Style Sheets en inglés) Es un lenguaje fácil de aprender, encargado de controlar la apariencia visual (diseño) de una página Web. Con este curso de CSS aprenderás, directo al grano, sin rodeos, LAS BASES DE CSS que realmente necesitas para dar el primer paso al diseñar una página Web desde cero, con una metodología de trabajo útil y fácil de entender; para que puedas organizar de manera óptima tu código CSS. Al final del curso, verás una introducción diagramando una página Web desde cero para que apliques todos los conocimientos aprendidos. Sin duda este curso de CSS abrirá tu mente al mundo de diseño y desarrollo Web.
